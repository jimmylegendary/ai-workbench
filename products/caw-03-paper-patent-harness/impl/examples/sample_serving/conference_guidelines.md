# Conference Guidelines — Systems for ML (sample)

- **Venue:** Workshop on Systems for Machine Learning (sample venue)
- **Page limit:** 8 pages (excluding references)
- **Mandatory sections:** Abstract, Introduction, Related Work, Method, Experiments, Conclusion
- **Format:** single-column LaTeX `article`, `natbib` citations, `plainnat` bibliography
- **Submission deadline:** 2026-09-15 (drives the literature temporal cutoff)
- **Reproducibility:** every reported number must trace to a result id in the experimental log; do not invent numbers
- **Scope:** inference serving, request scheduling, batching, latency/throughput trade-offs for LLM systems
